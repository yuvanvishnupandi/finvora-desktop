# Frequently Asked Questions (FAQ)

### Can I run Finvora without an internet connection?
Yes! The core tracking, budgeting, and dashboard features are entirely offline and data is saved securely to your local hard drive. However, the AI features (Voice Assistant, Receipt Scanner) require an active internet connection to contact external LLM APIs.

### Where is my data saved?
Data is saved locally inside the `expense-tracker-springboot-server/data/` folder as an H2 Database file.

### Is Finvora a website?
No, Finvora is a native Desktop Application built using JavaFX. It runs directly on your computer's operating system, yielding high performance and avoiding browser sandbox limitations.

### How do I export my data?
Navigate to the Settings or Dashboard view and click the **Export** button. You can export your data as a beautifully formatted PDF report.
